# Zunaid Hasan Tajim — Research Profile Website

A personal academic/research profile website for Zunaid Hasan Tajim,
Materials Science & Engineering Researcher, RUET Bangladesh.

## 📁 Project Structure

```
zunaid_website/
├── index.html          ← Main HTML page
├── css/
│   └── style.css       ← All styles & responsive design
├── js/
│   └── main.js         ← Scroll animations, counter, mobile nav
├── assets/
│   └── profile.jpg     ← Profile photo
└── README.md
```

## 🚀 How to Use

### Open Locally
Simply open `index.html` in any modern web browser. No server required.

### Deploy to GitHub Pages (Free Hosting)
1. Create a new GitHub repository (e.g. `zunaid-tajim.github.io`)
2. Upload all files maintaining the folder structure
3. Go to Settings → Pages → Source: main branch → /root
4. Your site will be live at `https://zunaid-tajim.github.io`

### Deploy to Netlify (Free)
1. Go to https://netlify.com and sign up
2. Drag & drop the entire `zunaid_website` folder onto the Netlify dashboard
3. Your site gets a live URL instantly

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Animated photo ring (CSS conic gradient)
- Scroll-triggered fade-in animations
- Animated stat counters
- Sticky navigation with active section highlight
- Mobile hamburger menu
- Back-to-top button
- All 4 publications with live DOI links
- Social profile links (LinkedIn, ResearchGate, Google Scholar, ORCID)
- IELTS score grid
- Contact section with reference supervisors

## 📝 Customization

- **Profile photo**: Replace `assets/profile.jpg` with an updated photo
- **Content**: Edit `index.html` directly — all content is clearly labelled
- **Colors**: Change CSS variables at the top of `css/style.css`
- **Fonts**: Loaded from Google Fonts (Playfair Display + Inter)
